Boom is our amazing PrestaShop theme, it’s specially designed with black color which would help you to form a unique looking website and have it stands out from the crowd.

Boom is perfectly suitable for fashion & accessories stores, it’s fully responsive and easy to install as well as to make customizations with a powerful theme editor panel. This theme is carefully developed and tested according to popular website standards today (w3c validation, all custom PrestaShop modules are validated, browsers compatibility, HTML/CSS3 implementation…)

 

Key features:

  • Fully responsive PrestaShop theme
  • Multi-language with backend translation management feature
  • Multi-currency (work with all national currencies in the world)
  • 6 home page layouts
  • 8 preset color skins/support custom color
  • 3 contact page layouts
  • 2 product page layouts
  • Powerful theme editor (with live editor)
  • One-click to import sample data
  • Support Google fonts
  • Rich mega menu module
  • Nivo slider with great text transition effects
  • Pretty Prestashop blog module
  • Widget module
  • Ajax search
  • Ajax cart
  • Social media module
  • Google map contact form
  • Manufacturer slider module
  • 1-page checkout
  • 5 steps checkout
  • Client testimonials
  • And many more…

 

NOTE:

The install package which you can download from Presta Hero is compatible with PrestaShop 1.7.x. If you want to download the install package for PrestaShop 1.6.x, please contact Presta Hero support staff.

Installation

Step 1: Log in to your back office, then navigate to Design > Theme & Logo

Step 2: Click on "Add new theme" button, then select your theme installation file (.zip file) and upload it.

Step 3: Choose the theme you uploaded and click “Use this theme”.

Step 4: Upload logo: Navigate to “Theme & Logo / Theme”, select and upload your website logo.
